# Local Photo Browser

ローカル環境に保存されている写真・動画を、ブラウザから閲覧するためのWebアプリです。
本アプリは localhost 上で動作し、外部サービスとの通信は一切行いません。

---

## 機能

- 指定フォルダ内の写真・動画をグリッド表示（無限スクロール・グリッド列数調整対応）
- 動画の再生（ブラウザネイティブ）、ポスターフレームのサムネイル・再生時間表示
- メディア種別フィルター（すべて／写真のみ／動画のみ）
- 写真の選択（最大4枚）と4枚ランダム表示（横並び・縦並びレイアウト切り替え）
- 日付・ファイル名・ランダムによる並び替え
- 単体ビューワ（前後ナビゲーション・ズーム・キーボード操作・フルスクリーン）
- ビューアからグリッドへ戻った際に、直前に開いた写真までスクロール位置を復元
- 全画面表示中のスクリーンショット保存（保存先フォルダは設定画面で指定）
- エクスプローラーで写真ファイルを直接開く
- サムネイル自動生成・キャッシュ
- フォルダツリーによるブラウジング（検索機能付き・配下フォルダの表示切り替え）
- スキャン対象フォルダの選択・除外設定（スキャン済みフォルダのバッジ表示）
- フォルダ単位でのスキャンデータ削除（DBレコード・サムネイルのみ削除し、ファイル本体は残す）
- スキャン・再スキャン・削除の進捗バー表示
- テーマ切り替え（システム追従・ダーク・ライト）
- 多言語対応（日本語・英語）
- 完全ローカル・オフライン動作

## 技術構成

### バックエンド
- **Python** + **FastAPI** + **Uvicorn**
- **SQLAlchemy 2.0**（同期）
- **SQLite**（WALモード）
- **Pillow**（サムネイル生成・画像処理）
- **imageio-ffmpeg**（動画のポスターフレーム生成・解像度/再生時間取得。ffmpegバイナリ同梱）

### フロントエンド
- **React 19** + **TypeScript**
- **Vite 6**（ビルド・開発サーバ）
- **React Router DOM 7**（ルーティング）
- **Zustand 5**（状態管理）

### データベース
- SQLite（`backend/data/app.db` に自動作成）
- テーブル：photos, person_tags, photo_persons, settings

---

## 対応形式

### 画像
- JPG / JPEG
- PNG
- WEBP

### 動画
- MP4 / M4V
- WEBM
- OGG / OGV
- MOV

※動画の再生可否はブラウザ・コーデックに依存します（MP4(H.264/AAC)・WEBM が確実）。

## 起動方法

### 方法1: バッチファイルで起動（Windows）

```
start-dev.bat
```

依存パッケージのインストールからバックエンド・フロントエンドの起動までを自動で行います。

### 方法2: Python スクリプトで起動

**開発モード**（フロントエンド開発サーバ付き）:

```
py scripts/start.py --dev
```

**本番モード**（ビルド済みフロントエンドをバックエンドから配信）:

```
py scripts/start.py
```

### 方法3: 手動で起動

バックエンドとフロントエンドをそれぞれ起動します。

```bash
# バックエンド依存パッケージのインストール
pip install -r backend/requirements.txt

# バックエンド起動（ポート8000）
py backend/main.py

# フロントエンド依存パッケージのインストール（別ターミナル）
cd frontend
npm install

# フロントエンド開発サーバ起動（ポート5173）
npm run dev
```

### アクセスURL

| 用途 | URL |
|------|-----|
| フロントエンド（開発時） | http://localhost:5173 |
| バックエンドAPI | http://localhost:8000 |
| APIドキュメント（Swagger UI） | http://localhost:8000/docs |

---

## 使い方

1. アプリを起動する
2. ブラウザで http://localhost:5173 を開く
3. 設定画面でルートフォルダを指定する（スクリーンショットを使う場合は保存先フォルダも設定）
4. スキャンを実行する
