# Local Photo Browser

ローカル環境に保存されている写真を、ブラウザから閲覧するためのシンプルなWebアプリです。  
本アプリは localhost 上で動作し、外部サービスとの通信は一切行いません。

---

## 機能

- 📁 指定フォルダ内の写真を閲覧
- 🎲 写真のランダム表示
- ⭐ お気に入り登録・管理
- 👤 人物タグによる分類
- 🔍 タグ・お気に入りによる絞り込み
- 🔄 日付・ファイル名・ランダムによる並び替え
- 🖥 完全ローカル・オフライン動作

---

## コンセプト

本アプリは、写真を気軽に眺めたり整理したりするための **個人向け写真ビューワ** として設計されています。

- クラウド連携なし
- 外部API利用なし
- 写真データはすべてローカル保存

プライバシーを重視した設計になっています。

## 使用方法（想定）

1. ローカルサーバを起動
2. ブラウザでアプリを開く
3. 写真フォルダを選択
4. 写真の閲覧を開始

---

## プライバシー

- すべてのデータはローカルに保存されます
- 外部ネットワーク通信は行いません
- アプリは `localhost` のみで動作します
